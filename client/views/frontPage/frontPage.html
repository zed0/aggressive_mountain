Welcome!

This is a quick demo of how to use Angular! Currently there is very little here, apart from an extremely basic todo app. You are encouraged to add to this site!

This is intended to be a collaborative project, so once you've made some changes, make a pull request, and we'll merge them in.
With any luck, we'll collaborate to make a decent site.

If you aren't sure what to do, or are having trouble doing what you want to, don't hesitate to ask!

To get started modifying this site take a look at the Getting Started page.

If you don't have any ideas for something to do here are some suggestions (roughly listed in easy to hard order):

  • Add some more fields to the todo list, for example deadlines (html/javascript)
  • Make it look prettier (html/css)
  • Make changes to the todo list persist to the browser(javascript)
  • Allow modifications to your existing items (javascript)
  • Add searching for items (javascript)
  • Add tags to items (javascript/html/css)
  • Add bootstrap to the site (html/css)
  • Add notifications (javascript)
  • Add linting to the project (javascript/CLI)
  • Add another app to the site (javascript/html/css)
  • Add jasmine tests to the project (javascript)
  • Add a build system to build the site (javascript/grunt/gulp/broccoli/brunch)
  • Make changes to the todo list persist to a server via REST (javascript/serverside language)
